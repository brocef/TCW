# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Fixed: TCW now runs on Python 3.11, 3.12, and 3.13

TCW advertises support for Python 3.11 and newer, but earlier versions actually
crashed on startup on Python 3.11, 3.12, and 3.13 — only Python 3.14 worked. That
is now fixed: TCW installs and runs on every supported Python version. If you
switched to Python 3.14 as a workaround, you no longer need to.
