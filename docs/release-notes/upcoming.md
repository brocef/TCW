# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Changed

- **Shorthand for effort and complexity.** `--effort` and `--complexity` (on
  `tcw work new` and `tcw work edit`) now also accept `L`, `M`, `H`, and `VH` as
  shorthand for `low`, `medium`, `high`, and `very-high` — in any capital-
  ization, so `--effort h` and `--complexity VH` both work. The full words keep
  working, and what gets recorded is always the full word.
