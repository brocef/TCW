# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Added

- **Reporting issues and suggestions just got easier.** A new guide walks you
  through sending feedback about TCW itself — a bug or a feature idea — as a
  GitHub issue, and gives you a ready-to-fill template for each. Bug reports ask
  for the essentials (your version, steps to reproduce, what you expected, and
  any fix you found); suggestions ask for the motivation, what you'd like, and
  who it helps. File anything at https://github.com/brocef/TCW/issues.
