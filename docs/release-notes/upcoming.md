## Filter the web board by status

The `tcw serve` work board now has a row of status toggle buttons above the list —
one each for **inbox**, **backlog**, **active**, and **completed**. Turn a status
on to show its items; **completed items are hidden by default** so the board leads
with what's still in flight. The toggles work alongside the existing text filter.
