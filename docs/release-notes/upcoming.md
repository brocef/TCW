# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Web app

- The object list in the web app is now a **collapsible tree** that mirrors
  each section's hierarchy: taxonomy terms and capabilities nest by their
  paths, and work items nest under their parent items. Grouping labels that
  aren't items themselves show as plain folders you can expand and collapse.
  Picking or linking to a nested item automatically opens the folders above
  it, and the search box narrows the tree to matches while keeping them
  reachable. The work list no longer separates items under status headings —
  the status filter buttons and the colored status label on each row still
  work as before.
