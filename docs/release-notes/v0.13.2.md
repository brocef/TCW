# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Improved

- Descendant work boards now indent independently scheduled tasks beneath their
  owning epic, including tasks stored in another registered project.
- `tcw work list` now accepts `-i` and `--incl-desc` as shorter alternatives to
  `--include-descendants`.
- Planning and backlog-audit guidance now keeps work-item tags useful and
  current, using the project's existing registered tag vocabulary.
