# Upcoming

User-facing release notes for the next version. Plain language — no jargon or
internal module names.

## Fixes

- Installed copies now include the current web interface bundle, including the
  staged-plan controls already available from source.

## New features

- The local web app now has Light, Dark, and System appearance choices under a
  Settings gear after Work. System is the default, follows operating-system
  changes, and the choice stays in the current browser.
- The complete web interface now uses a consistent, compact visual system for
  browsing, editing, dialogs, filters, validation, and conflict recovery.
